# Project Name

Personal Notes App

# Project Description

Personal Notes App is a web application that allows users to securely create, manage, update, and delete personal notes.

Users can register an account, log in securely using JWT authentication, upload a profile picture, and manage their notes from a dashboard.

# The application demonstrates full-stack development using:

React + TypeScript
FastAPI
PostgreSQL
Main Objective

# The primary objective of this project is to learn:

Frontend–Backend communication
REST API development
Authentication and authorization
Database integration
File upload handling
Full-stack deployment
